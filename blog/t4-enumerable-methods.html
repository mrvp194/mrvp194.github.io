The Cycle Enumerable

11/2/2014

Enumerable#cycle can be called on any enumerable object (a range, hash, or array)Cycle calls a block of code that you provide for each element however many times you specify. If you do not choose the amount of times to run the block, it will repeat forever. You can designate a break to prevent this from happening.

array = ["Mom", "Dad", "Brother", "Sister"]
array.cycle(3) {|x| puts x}
==> Mom, Dad, Brother, Sister, Mom, Dad, Brother, Sister, Mom, Dad, Brother, Sister

array.cycle {|x| puts x}
==> Mom, Dad, Brother, Sister, Mom, Dad, Brother, Sister...repeats forever
